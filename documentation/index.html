Dependency

The template made with some jQuery plugin to give a nice look and functionality. We use Google Font to make the template. You must have to keep these jquery plugins if you wish the template work same to you as preview. 

 

The template created with the help of: 

  • Twitter Bootstrap 3x
  • Google Font
  • Jquery
  • Google Map API v3
  • jQuery Plugins
    • Bootstrap Select
    • Owl Carousel
    • Revolution Slider
